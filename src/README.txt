Orientação sobre diretórios:
* 'node' - códigos da estrutura base da árvore de busca
* "algorithms' - códigos de todos os algoritmos requeridos pelo Trabalho


Para executar qualquer código/comando, execute no diretório './src'.

Para compilar:
$ make

Para executar algum algortimo específico:
$ ./busca.sh <nome_do_algortimo> <diretorio_do_input>
